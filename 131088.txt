변수

파이선에서는 변수는 문자, 숫자 밑줄을 포함할 수 있다.
단, 숫자가 맨 앞에 오지는 못한다.
대소문자 구별에 주의해야 한다.

False, None, True, and, as등은 예약어로서 변수명으로 사용이 불가능하다.


조건문

if문

파이썬은 들여쓰기가 엄청 중요
C언어에선 블럭을 형성할 땐 중괄호를 사용하지만 파이썬에선 들여쓰기로 검사

if 문의 기본 구조

if (조건식) :
	<조건이 맞을 경우 실행>

if옆 소괄호 안에 들어있는 조건식에 어떠한 조건이 만족된다면 if문 블럭 안에 있는 명령들은 실행이 되는 것을 말한다.

elif 문

C언어로 치면 else if 를 뜻하는 말로 위에서부터 차례대로 검사하되 하나가 만족할 경우 현재 검사 중인 if~elif 문을 pass하게 할수 있다.

elif 의 기본 구조는 if와 마찬가지로 아래와 같다.

if (조건식) :
	<조건이 맞을 경우 실행>
elif (조건식) :
	<if에 실행 X elif 조건이 성립할 때 실행>


else문

위의 if 와 elif 에서 명시되었던 조건들이 전부다 만족하지 않았을 경우 그 외의 경우의 모든 조건을 말하는 명령어

기본 구조는 아래와 같다.

if (조건식) :
	<조건이 맞을 경우 실행>
elif (조건식) :
	<if에 실행 X elif 조건이 성립할 때 실행>
else :
	<디폴트 실행>

연산자

연산자(Operators)는 덧셈과 곱셈 같은 계산을 표현한다. 연산자가 사용하는, 연산에 사용되는 값을 피연산자(operands)라고 칭한다.

 +, -, *, /, 그리고 그룹화를 위한 괄호의 사용은, 수학에서 뜻하는 것과 같다.

변수 이름이 피연산자의 위치에 쓰일 때 변수명은 연산이 수행되기 전에 그 값으로 대체된다.
